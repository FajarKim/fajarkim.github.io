Enkripsi File Shell dengan BZip2 Shell Exec

Mengenal Apa Itu BZip2 Shell Exec?

Kata "Exec" dalam nama alat ini merupakan singkatan dari kosakata bahasa Inggris, yaitu executable. Dalam konteks dunia pemrograman, executable adalah file yang berisi instruksi-instruksi yang dapat dieksekusi atau dijalankan oleh sistem operasi. Biasanya, executable digunakan untuk menjalankan program atau aplikasi komputer.

Executable biasanya dibangun dari kode sumber program yang telah dikompilasi menjadi bentuk yang dapat dieksekusi oleh sistem operasi. Proses kompilasi mengubah kode sumber program (biasanya ditulis dalam bahasa pemrograman tingkat tinggi seperti C, C++, Java, Python, dll.) menjadi instruksi mesin yang dapat langsung dijalankan oleh komputer.

Jadi, BZip2 Shell Exec ini merupakan alat yang dirancang khusus, dimana ketika dijalankan, alat ini akan mengenkripsi atau mengompres semua string file shell ke dalam format BZip2 tanpa peru khawatir direcode atau diambil orang lain. File yang sudah dienkripsi akan tetap bisa dijalankan dengan perintah bahasa shell. Adapun jenis bahasa shell yang didukung adalah Bourne Shell (sh), Bourne Again Shell (bash), Z Shell (zsh), Korn Shell (ksh), dan MirBSD Korn Shell (mksh).

Petunjuk Penginstalan

Termux

Terdapat dua cara untuk menginstalnya di aplikasi Termux, yaitu:

Cara 1

Cara ini merupakan cara yang manual dan cukup ribet. Caranya cukup dengan menjalankan beberapa perintah yang diperlukan. Berikut ini beberapa perintah yang perlu dijalankan!

  • pkg update -y && pkg upgrade -y
  • pkg install git bzip2 xz-utils -y
  • git clone https://github.com/FajarKim/bz2-shell
  • cd bz2-shell

  • Kemudian, jalankan file bzsh.sh untuk mulai mengenkripsi file shell.

    bash bzsh.sh --help

  • Jika Anda ingin menginstalnya ke folder $PATH, cukup jalankan file install.sh yang ada di folder tools/Termux.

    tools/Termux/install.sh

    atau

    cd tools/Termux && bash install.sh

  • Jika berhasil terinstal, jalankan perintah:

    bzsh --help

Cara 2

Sedikit berbeda dengan cara sebelumnya, cara ini bisa dibilang simpel karena akan langsung menginstal alat ini ke folder $PATH. Berikut ini caranya!

  • pkg update -y && pkg upgrade -y
  • pkg install git bzip2 xz-utils -y

  • Jika paket sudah terinstal, langkah selanjutnya dengan menjalankan perintah berikut ini!

    pkg install curl -y
    bash -c "$(curl -fsSL https://raw.githubusercontent.com/FajarKim/bz2-shell/master/tools/Termux/install.sh)"
    pkg install wget -y
    bash -c "$(wget -qO- https://raw.githubusercontent.com/FajarKim/bz2-shell/master/tools/Termux/install.sh)"
    pkg install fetch -y
    bash -c "$(fetch -o - https://raw.githubusercontent.com/FajarKim/bz2-shell/master/tools/Termux/install.sh)"

    Sebagai alternatif, Anda dapat mengunduh skrip install.sh terlebih dahulu dan menjalankannya setelah menginstalnya:

    • pkg install wget -y
    • wget https://raw.githubusercontent.com/FajarKim/bz2-shell/master/tools/Termux/install.sh
    • bash install.sh

      Petunjuk Penggunaan

      Alat ini diprogram untuk bisa dijalankan dengan beberapa perintah. Perintah yang didukung:

      Perintah
      Keterangan
      -h atau --help Menampilkan bantuan untuk alat ini
      -v atau --version Informasi versi keluaran terbaru
      -t atau --type-shell Pilih jenis shell (sh, bash, zsh, ksh, atau mksh)
      -f atau --file Kompres setiap FILE sebagai gantinya

      Cara menjalankan alat ini:

      Contoh 1

      bzsh -t bash -f FILE

      atau

      bzsh --type-shell bash --file FILE

      Contoh 2

      Jika ingin mengenkripsi lebih dari 1 file dalam waktu yang sama, Anda bisa lakukan itu.

      bzsh -t bash -f FILE1 FILE2 FILE3 etc...

      atau

      bzsh --type-shell bash --file FILE1 FILE2 FILE 3 etc...

      Catatan

      Semakin banyak jumlah baris dalam sebuah file atau semakin besar ukuran sebuah file, semakin lama waktu yang dibutuhkan untuk mengenkripsi.

      Alat ini mungkin tidak kompitabel atau tidak didukung dibeberapa Linux, seperti di Ultrix.
